# Learnova Internship Tasks

## Overview
This repository contains the projects completed as part of the Learnova Internship Program. The tasks were developed using HTML, CSS, and JavaScript while focusing on responsive design, user experience, and front-end development concepts.

## Projects Included

### 1. Portfolio
A responsive portfolio website showcasing my skills, education, projects, and career interests as a Computer Engineering student.

#### Features
* Responsive design
* About Me section
* Skills section
* Education section
* Projects showcase
* Contact section

### 2. Temperature Converter Website
A web application that converts temperature values between Celsius, Fahrenheit, and Kelvin.

#### Features

* Input validation
* Multiple temperature unit conversions
* Interactive user interface
* Instant result display

### 3. Tribute Page
A tribute webpage dedicated to Grace Hopper, highlighting her achievements and contributions to computer science.

#### Features
* Biography section
* Achievements section
* Responsive design
* Modern layout

## Technologies Used
* HTML5
* CSS3
* JavaScript
